== Description ==

A simple plugin that allows for users to embed "Realtime Search" results into posts and pages using a shortcode. Results are aggregated from Twitter, Facebook, and Google Blog Search. Future version will include more networks and the ability to customize the design.

[youtube http://www.youtube.com/watch?v=3AESuLN5a70]

== Installation ==

1. Upload the entire `social_search` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[Social Search q="search term"]` in your post or pages where you want results to appear.
